# 🔐 Secure Password Generator

A sleek, modern, and highly customizable random password generator built with React and Tailwind CSS. Designed with a premium "glassmorphism" UI, this tool allows users to instantly generate secure passwords tailored to their specific requirements.

## ✨ Features

* **Real-time Generation:** Passwords generate automatically as you adjust the settings—no need to constantly click a button.
* **Customizable Length:** Granular control over password length via a smooth range slider (from 6 up to 32 characters).
* **Character Toggles:** Easily include or exclude numbers and special symbols to meet varying security requirements.
* **One-Click Copy:** Instantly copy the generated password to your clipboard with built-in visual confirmation and text highlighting.
* **Premium UI/UX:** Features a responsive, dark-mode-first design with gradient typography, smooth hover animations, and dynamic state changes.

## 🛠️ Tech Stack

* **Frontend:** React.js (Hooks: `useState`, `useEffect`, `useCallback`, `useRef`)
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
